Gartenplaner PWA Setup

Klicke auf die Buttons, um die Dateikomponenten herunterzuladen:

Anleitung:
1. Alle 3 Dateien in denselben Ordner legen.
2. Ein Bild namens icon.png (192x192px) in den Ordner legen.
3. Die index.html über einen lokalen Webserver öffnen (z.B. mit der VS Code Erweiterung "Live Server"), damit die PWA-Funktionen aktiv werden.
